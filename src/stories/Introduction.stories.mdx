# React Floating Whatsapp

Simple React component for adding a floating WhatsApp button to your project.

[![NPM](https://img.shields.io/npm/v/react-floating-whatsapp.svg)](https://www.npmjs.com/package/react-floating-whatsapp) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)![npm bundle size](https://img.shields.io/bundlephobia/min/react-floating-whatsapp)![GitHub](https://img.shields.io/github/license/awran5/react-floating-whatsapp)

---

##### Install with npm

```sh
npm i react-floating-whatsapp

```

##### Yarn

```sh
yarn add react-floating-whatsapp
```
